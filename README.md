# Willow Wisk

A fully custom Discord bot designed for community management, ticket systems, verification, order logging, embeds, moderation, automod, economy, backups, and dashboard management.

## Planned Systems

- Dashboard
- Verification System
- Ticket Panels
- Welcome & Leave Messages
- Order Logging
- Automod
- Moderation Commands
- Reaction Roles
- Server Backup
- Economy System
- Analytics

Theme:
Soft greens, browns, cream, muted teal.
